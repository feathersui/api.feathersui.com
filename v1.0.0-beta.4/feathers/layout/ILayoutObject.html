Variables

@:flash.propertyincludeInLayout:Bool

Determines if the ILayout of the parent container should measure and position this object or ignore it.

When the value of the includeInLayout property changes, the object will dispatch an event of type FeathersEvent.LAYOUT_DATA_CHANGE.

In the following example, the object is excluded from the layout:

object.includeInLayout = false;
Available since

1.0.0

.

See also:

  • feathers.events.FeathersEvent.LAYOUT_DATA_CHANGE

@:flash.propertylayoutData:ILayoutData

Optional, extra data used by some ILayout implementations.
