class TriggerEvent

package feathers.events

extends Event

@:directlyUsed

Represents a click, tap, or triggering key press dispatched by button components.

Available since

1.0.0

.

Static variables

@:value("trigger")staticinlineread onlyTRIGGER:EventType<TriggerEvent> = "trigger"

The TriggerEvent.TRIGGER event type is dispatched when a Feathers UI component is clicked or tapped.

Available since

1.0.0

.
