interface ITriggerView

package feathers.controls

extends IUIControl

extended by BasicButton, BasicToggleButton

@:directlyUsed@:event(feathers.events.TriggerEvent.TRIGGER)

A UI component that dispatches TriggerEvent.TRIGGER. Some components may prefer to listen for TriggerEvent.TRIGGER instead of MouseEvent.CLICK or TouchEvent.TOUCH_TAP.

Events:

feathers.events.TriggerEvent.TRIGGER

Dispatched when the the user triggers the view.
