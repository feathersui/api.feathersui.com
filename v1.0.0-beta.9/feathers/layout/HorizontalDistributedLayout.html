class HorizontalDistributedLayout

package feathers.layout

extends EventDispatcher

implements ILayout

@:directlyUsed@:event(openfl.events.Event.CHANGE)

Positions items from left to right in a single row, and all items are resized to have the same width and height.

Events:

openfl.events.Event.CHANGE

Dispatched when a property of the layout changes, which triggers the container to invalidate.

Available since

1.0.0

.
